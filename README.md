Student Attendance Management System
Description

Le Student Attendance Management System est une application web conçue pour gérer la présence des étudiants de manière simple et efficace.
Le système est organisé en trois portails distincts : Admin, Teacher et Student, chacun avec des fonctionnalités spécifiques.

Portails
Admin Portal

Gérer les enseignants et les étudiants

Créer et gérer les classes et les matières

Consulter et gérer les statistiques de présence

Teacher Portal

Marquer la présence des étudiants

Consulter l’historique des présences

Gérer ses classes et matières

Student Portal

Consulter sa présence

Voir son historique de présence

Objectif

Ce projet vise à automatiser la gestion des présences, réduire les erreurs manuelles et faciliter le suivi pour l’administration, les enseignants et les étudiants.
